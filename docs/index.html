Built-in local webserver overlay

Smart Lower Thirds runs a small local webserver that serves your overlay as a persistent Browser Source, keeping your graphics in sync across every scene.

Persistent Browser Source

Once configured, the plugin automatically provides a Browser Source URL you can reuse across scenes:

http://127.0.0.1:8089/smart-lower-thirds.html

Add a Browser Source once, then reuse it in all your scenes. When you update a lower third, the overlay updates everywhere‚Äîno extra wiring needed.

  • Works fully offline
  • No third-party hosting
  • Perfect for secure or studio environments

Dock Interface

Manage all your lower thirds and plugin settings from the dedicated Smart Lower Thirds dock inside OBS Studio.

Settings Dialog

Fine-tune typography, animations, colors, avatars, and behavior using the fully featured settings dialog.

Powerful placeholder system

Design your own HTML/CSS templates using simple, readable placeholders that Smart Lower Thirds replaces at runtime.

Available placeholders

These can be used in both HTML templates and CSS:

Placeholder Description
{{TITLE}} Main text such as name, topic, or headline
{{SUBTITLE}} Secondary text, role, or description
{{PROFILE_PICTURE}} Avatar image file name / URL
{{ID}} Unique instance ID used for styling & animations
{{LT_POSITION}} Selected position class (bottom-left, top-right, etc.)
{{BG_COLOR}}, {{TEXT_COLOR}} Dynamic colors injected from the settings dialog

This system makes it easy to share templates, restyle overlays, or integrate Smart Lower Thirds into your existing brand.

Hotkeys & scene binding

Smart Lower Thirds gives you full control over when each lower third appears through OBS hotkeys and scene-based automation.

Hotkey integration: assign OBS hotkeys per lower third to show, hide, or toggle animations. Works great with Stream Decks, macro pads, or any control surface supported by OBS.

Scene binding: automatically show a specific lower third when entering a scene and hide it when leaving.

To keep overlays clean, triggering a lower third via hotkey or scene binding will hide all other active lower thirds so the focus stays on the one that is bound.

If you want multiple lower thirds visible at the same time (e.g., host + guest + topic), you can enable them manually from the dock.

Template import & export

Share your lower thirds, reuse setups between projects, and build your own template library using minizip `.zip` bundles.

Portable by design

Each template can contain:

  • HTML layout

  • Font and behavior settings

Export them, version them with git, share them with your team, or sell them in your overlay shop.

Installation

Installing Smart Lower Thirds is straightforward. Once installed, everything is managed directly inside OBS Studio.

Quick start

  1. Download the plugin for your platform from the GitHub releases page.
  2. Run the installer or copy the plugin files into your OBS plugins folder.
  3. Restart OBS Studio.
  4. Enable the Smart Lower Thirds dock from the Docks menu.
  5. Add a Browser Source pointing to http://127.0.0.1:8089/smart-lower-thirds.html.
  6. Create your first lower third and trigger it with a hotkey.
