What this plugin does
vinci-flow (VinciFlow) generates and drives HTML-based lower thirds for OBS Studio. It manages an OBS Browser Source, writes the live overlay HTML to disk, and lets you control visibility from the dock UI, hotkeys, or obs-websocket vendor requests.

Key concepts

Lower thirds

Each lower third is a configurable unit with an id, content (title, subtitle, optional avatar), styling (colors, radius, opacity), animation in/out, and optional sound cues. Internally, the overlay is composed from HTML/CSS/JS templates and placeholder substitution.

Groups

Groups orchestrate a list of lower thirds over time (linear or randomized), with timing controls and optional exclusivity. This is useful for sponsor rotators, team lineups, event speakers, or “now playing” style cycles.

Overlay output

  • The plugin writes one active HTML overlay file into your configured output directory.
  • Old overlay files matching lt-*.html are periodically cleaned up (only the current one is kept).
  • The Browser Source (typically named VinciFlow) points to the current overlay and is updated by the plugin.

Placeholders available to theme authors

These placeholders are substituted into your templates at render time: 

{ID}
{TITLE}          {TITLE_SIZE}        {TITLE_COLOR}
{SUBTITLE}       {SUBTITLE_SIZE}     {SUBTITLE_COLOR}
{PROFILE_PICTURE_URL}  {AVATAR_WIDTH}  {AVATAR_HEIGHT}
{PRIMARY_COLOR}  {SECONDARY_COLOR}
{OPACITY}        {RADIUS}
{FONT_FAMILY}    {ANIM_IN}           {ANIM_OUT}
{SOUND_IN_URL}   {SOUND_OUT_URL}

Legacy aliases: {BG_COLOR}, {TEXT_COLOR}
