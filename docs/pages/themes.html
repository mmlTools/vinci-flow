Theme authoring model

A “theme” is effectively the combination of:

  • HTML template (rendered inside <li id="{{ID}}">)
  • CSS template (ideally scoped to #{{ID}})
  • JS template (executed with root bound to the element for this lower third)
Scoping rule of thumb
Write CSS as if multiple lower thirds can co-exist: prefix selectors with #{{ID}} (or #{{ID}} .child) to avoid bleeding into other items.

Placeholders

Your templates can reference placeholders which are substituted at render time: 

{{ID}}
{{TITLE}} / {{SUBTITLE}} / {{PROFILE_PICTURE_URL}}
{{PRIMARY_COLOR}} / {{SECONDARY_COLOR}} / {{TITLE_COLOR}} / {{SUBTITLE_COLOR}}
{{OPACITY}} / {{RADIUS}} / {{FONT_FAMILY}}
{{TITLE_SIZE}} / {{SUBTITLE_SIZE}} / {{AVATAR_WIDTH}} / {{AVATAR_HEIGHT}}
{{ANIM_IN}} / {{ANIM_OUT}}
{{SOUND_IN_URL}} / {{SOUND_OUT_URL}}

Animations

The plugin supports two animation strategies:

1) Animate.css classes

  • Set anim_in and anim_out to Animate.css class names (e.g. animate__fadeInUp).
  • The placeholders {{ANIM_IN}} and {{ANIM_OUT}} resolve to the selected classes.

2) Custom-handled animations

If you set:

  • anim_in = "custom_handled_in"
  • anim_out = "custom_handled_out"

…then the plugin does not apply an Animate.css class and expects your JS template to provide: 

// Your JS template runs with:
const root = document.getElementById("{{ID}}");

// Optional: called when showing
root.__slt_show = function(){
  // start your custom "in" animation here
};

// Required for custom out: must return a Promise that resolves when you are done
root.__slt_hide = function(){
  // start your custom "out" animation and resolve when complete
  return new Promise((resolve) => setTimeout(resolve, 3000));
};

Sound cues

  • You can define optional sound files for “in” and “out”.
  • The placeholders {{SOUND_IN_URL}} and {{SOUND_OUT_URL}} allow your theme to play them.

Recommended theme structure

your-output-dir/
  assets/
    theme.css
    theme.js
    images/
  (plugin-generated) lt-*.html
Browser Source caching
While iterating on theme files, you may need to refresh the Browser Source to pick up changes.
